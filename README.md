# Descripción
Juego simple creado en Unity. Scroller lateral infinito en la que un aguila debe disparar o esquivar pajaros. 


# Controles: 
 	- botón flecha arriba para ir hacia arriba.
	- botón flecha abajo para ir hacia abajo.
	- barra espaciador para disparar.

# Problemas:
	- Cuando el jugador choca su águila con un pájaro, el águila desaparace. En teoría, se tendría que bajar la vida del jugador y el pájaro tendría que seguir su camino.
	- La barra de vida se desplaza hacia la izquierda al bajar la vida. No tendría que moverse.
